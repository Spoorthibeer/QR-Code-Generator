QR Code Label Printer for Inventory Management
Project Overview
This is a browser-based utility application designed to streamline the process of generating high-density, 
printable QR Code labels directly from an Excel spreadsheet (XLSX/XLS).
The system allows users to upload a spreadsheet containing multiple inventory records and instantly 
generates formatted labels ready for printing.
Key Features
Excel Data Ingestion: Reads and parses data from uploaded .xlsx or .xls files using a third-party library.
Dynamic QR Code Generation: Generates individual QR codes for each inventory record, encoding all row data into the code.
Print-Optimized Labels: Creates structured, printable labels with fields for MDB PARTNO, QTY, DATE CODE, and QR Code.
Clean Front-End: Utilizes HTML and CSS for a straightforward user interface and print media queries for zero-margin, paper-friendly output
